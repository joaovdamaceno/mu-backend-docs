---
title: Usar no IntelliJ
---

Você pode editar/rodar Docusaurus no IntelliJ como um projeto normal.

## Passo a passo

1. **File → Open** e selecione a pasta do projeto `mu-backend-docs`.
2. Abra o **Terminal** do IntelliJ.
3. Rode:

```bash
npm install
npm run start
```

4. Acesse: http://localhost:3000

Dica: se o terminal do IntelliJ não reconhecer `npm`, instale Node.js (LTS) e reinicie o IntelliJ.
